666 Luftballons
===============

Entry in PyWeek #4  <http://www.pyweek.org/4/>
Team: Gin 'n Pythonic
Members: Thijs Jonkman, Jeroen Vloothuis


DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/


RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.py" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

Press H on startup screen for help. You can also use the arrow keys to
highlight the help button and then press space.
